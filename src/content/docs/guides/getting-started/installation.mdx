    <Tabs>
      <TabItem label="Node.js, Deno, Bun">
        {/* Keep this empty, so it says that we're good to go! */}
      </TabItem>
      <TabItem label="React Native">
        This library uses [`react-native-real-fetch`](https://github.com/Vexcited/react-native-real-fetch) under the hood to make HTTP requests,
        this is why you're required to also install the following native modules.

        <Tabs>
          <TabItem label="Expo">
          ```bash
          npx expo add react-native-real-fetch react-native-nitro-modules
          npx expo prebuild
          ```
          </TabItem>
          <TabItem label="Bare">
          ```bash
          npm add react-native-real-fetch react-native-nitro-modules
          cd ios && pod install
          ```
          </TabItem>
        </Tabs>
      </TabItem>
      <TabItem label="Tauri">
        You should setup [Tauri's HTTP plugin](https://tauri.app/plugin/http-client/) and add the following allowed URL to the permissions.
        ```json
        {
          "permissions": [
            {
              "identifier": "http:default",
              "allow": [{ "url": "http://webservices-v2.crous-mobile.fr/feed/*" }],
            }
          ]
        }
        ```
      </TabItem>
    </Tabs>
  </TabItem>
  <TabItem label=".rs">
  ```bash
  cargo add crowous
  ```
  </TabItem>
</Tabs>

You're ready to go!
